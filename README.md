# Beyond Frozen Intelligence — Self-Evolving Machine Research Website

**Authors:** Kazi Mahir Adeeb (Lead) · S M Shadman Khan · Abuhena Shuvro  
**Domain:** AI & Physics Research · 2025

## About

This is the official research website for the paper:  
**"Beyond Frozen Intelligence: Adaptive Efficiency and the Self-Evolving Machine — A Unified Physical-Computational Framework for Cognitive Self-Optimization at the Intersection of Artificial Intelligence and Physics"**

## Features

- **Interactive 3D Machine Viewer** — Full 360° Three.js visualization of the SEM architecture. Click any component, rotate, zoom, watch live data flow.
- **Live Stochastic Simulation** — Real-time AE Growth Model with adjustable parameters. Watch the theorem's failure regimes activate in real-time.
- **Theory Deep-Dive** — Multi-audience explanations (Expert / Student / Teen) for every concept.
- **Full Paper** — Complete paper with all equations, tables, charts rendered beautifully.
- **Equations Reference** — Every equation from the paper with variable definitions.
- **Phase Portrait** — Interactive AE phase portrait showing growth vs failure regions.

## Deploy to GitHub Pages

1. Create a new repository (e.g., `sem-research`)
2. Push all files to the `main` branch
3. Go to Settings → Pages → Source: main branch
4. Site will be live at `https://yourusername.github.io/sem-research/`
