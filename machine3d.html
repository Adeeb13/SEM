Core Definitions
Def 1.1
Adaptive Efficiency
AE(t) = dC(t) / dt
The instantaneous rate of change of the Capability Function. Positive AE = the system is growing. Zero AE = stasis. Negative AE = regression. Unlike thermodynamic efficiency (bounded by Carnot), AE is not bounded above for open systems.
AE(t)C(t) = capabilityt = time/cycle
Def 2.1
Capability Function
C(t) = α · C_cog(t) + β · C_phys(t)
Weighted composite of cognitive and physical capability, measured on a dynamic benchmark suite B(t) that advances with the agent. α = 0.70 (cognitive primacy), β = 0.30. The dynamic benchmark prevents benchmark saturation (Ablation Table).
α = 0.70β = 0.30α + β = 1C_cog = cognitive scoreC_phys = physical score
Def 3.1
Effective Improvement Rate
λ_eff = α · λ_cog + β · λ_phys
The weighted average improvement rate across cognitive and physical dimensions. In the paper baseline: λ_cog = 0.050, λ_phys = 0.030, giving λ_eff = 0.044. The gradient of the AE phase portrait is steeper in λ_cog by factor α/β = 7/3.
λ_cog = 0.050λ_phys = 0.030λ_eff = 0.044
Stochastic Model
Eq 4.1
Stochastic Capability Update
C(t+1) = C(t) + F(t)·[λ_eff·C(t)·(1−C(t)/C_max) − δ_K(t)] + σ·C(t)·ξ(t)
The core stochastic differential equation governing SEM dynamics. Four physical effects: logistic bounded growth (Verhulst term), Kolmogorov incompressibility correction, Bernoulli failure events, and multiplicative Gaussian noise. Analogous to the Langevin equation in statistical physics.
ξ(t) ~ N(0,1) F(t) ~ Bernoulli(1−p_fail) σ = 0.08 p_fail = 0.07 C_max = 1000 (baseline) δ_K(t) = incompressibility correction
Eq 4.2
Expected AE
E[AE(t)] = (1−p_fail)·[λ_eff·C(t)·(1−C/C_max) − δ_K(t)] − p_fail·μ_fail·C(t)
Derived by taking the conditional expectation over F(t) and ξ(t). This is the master equation from which the theorem's conditions follow directly. Setting this expression > 0 gives the necessary condition.
μ_fail = expected fractional capability loss per failure
Adaptive Efficiency Growth Theorem
Theorem N
Necessary Condition for E[AE(t)] > 0
λ_eff · C(t) · (1 − C(t)/C_max) > δ_K(t) + p_fail · μ_fail · C(t)
Growth rate × available headroom must exceed incompressibility cost + expected failure loss. This is a point-in-time condition: as C(t) increases toward C_max, the left-hand side shrinks while δ_K(t) grows — eventually violating the condition (FR-2/FR-3).
Theorem S
Sufficient Condition for Sustained Positive AE
λ_eff > (σ² / 2) + p_fail · μ_fail + δ_K_max / C_0
The noise drift term σ²/2 arises from Itô's lemma applied to the multiplicative noise σ·C(t)·ξ(t). This is the condition that, if satisfied at t=0 with adequate headroom, guarantees sustained positive AE over [0,T]. Baseline: 0.044 > 0.0032 + 0.0042 + 0.001 ✓
δ_K_max = max δ_K(t) over [0,T]C_0 = C(0) = 8.5
Failure Regimes
FR-1
Noise Dominance
σ²/2 > λ_eff − p_fail·μ_fail
When stochastic fluctuations in the modification-evaluation cycle overwhelm the systematic improvement signal. Physical analog: order-disorder phase transition in spin glasses. The critical noise threshold is σ²/2 = λ_eff − p_fail·μ_fail.
FR-2
Resource Saturation
C(t) → C_max ⟹ AE(t) → −δ_K(t) < 0
As C(t) approaches C_max, the logistic growth term approaches zero while δ_K(t) continues to grow — eventually driving AE negative. Physical analog: carrying capacity limit (logistic equation / Verhulst, 1838).
FR-3
Algorithmic Incompressibility
δ_K(t) ≥ λ_eff · C(t) · (1 − C(t)/C_max)
The deepest failure mode. When K(A_t) — the Kolmogorov complexity of the agent description — grows faster than the modification strategy can discover improvements. Escaping requires second-order self-improvement (improving the modification strategy itself).
Physical Foundation Equations
Physics 1
Carnot Efficiency Limit
η_Carnot = 1 − (T_cold / T_hot)
The thermodynamic efficiency ceiling for any heat engine. AE is in a categorically different domain — rate of capability change for open systems — and is not bounded by any analogous Carnot-like ceiling.
Physics 2
Landauer's Principle
E_erase ≥ k_B · T · ln(2) ≈ 2.87 × 10⁻²¹ J (at T = 300K)
Minimum energy to erase one bit of information. Sets a physical floor on the cost of every self-modification cycle. Motivates the DGM archive strategy: retaining rather than erasing failed modifications avoids this cost.
k_B = 1.38×10⁻²³ J/KT = temperature (Kelvin)
Physics 3
Kolmogorov Complexity Bound
K(s) = length of shortest program producing s
No program produces string s with fewer bits than K(s). Applied to the agent's cognitive state A_t: K(A_t) increases monotonically as capability grows. This is the information-theoretic underpinning of FR-3.
Physics 4
Entropy Production (Prigogine)
dS_universe/dt = dS_system/dt + dS_environment/dt ≥ 0
Total entropy production is always non-negative (Second Law). The SEM as a dissipative structure: it increases internal organization (dS_system < 0 locally) by exporting entropy to the environment at a greater rate.
AE Phase Portrait
Eq 8.1
AE = 0 Boundary in (λ_cog, λ_phys) Space
α·λ_cog + (1−α)·λ_phys = (σ²/2) + p_fail·μ_fail + δ_K/C
The critical curve separating the sustained growth region from the failure region in parameter space. The gradient of this boundary is steeper in the λ_cog direction by factor α/β = 7/3 — motivating prioritization of cognitive optimization in resource-constrained implementations.
Gradient: ∂(boundary)/∂λ_cog = α/β = 7/3
